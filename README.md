# Tic-Tac-Toe

A simple, interactive **Tic-Tac-Toe** game built with python. This project demonstrates basic game logic, state management, and user interface design.

##  Features
* **Two-player mode:** Play locally with a friend.
* **Interactive Interface:** Clean and easy-to-use game board.
* **Win Detection:** Automatically detects wins, losses, and draws.
* **Reset Functionality:** Quickly start a new game after a round ends.
